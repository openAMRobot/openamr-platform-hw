# Security Policy

## Reporting Security Issues

Please do not report security issues publicly in GitHub Issues.

If you discover a safety-relevant error in the hardware documentation (wiring, power,
grounding, battery/charging, or a value that could damage hardware or injure someone),
contact the maintainers privately:

    botshare.ai@gmail.com

Include: repository name, affected file, a description, and the potential impact.

## Scope

This repository is documentation. "Security" here means **safety-relevant correctness**:
wiring/pinouts, voltage rails, power distribution, battery/charging, and the documented
safety gaps. See the "Known hardware limitations" section of the README.
